# 工具脚本

## 目的

存放数学建模中的辅助工具脚本（不属于算法库的小工具）。

## 规划工具列表

| 工具 | 文件 | 功能 |
|------|------|------|
| CSV→LaTeX 表格 | `csv_to_latex.py` | 将 CSV/Excel 数据转为 LaTeX 表格代码 |
| 论文查重辅助 | `duplicate_check.py` | 本地查重预处理 |
| 数据预览 | `data_explorer.py` | 快速了解数据基本特征 |
| 结果格式化 | `result_formatter.py` | 将模型输出格式化为论文可用的样式 |
