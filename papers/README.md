# 优秀论文分析

## 目的

分析历年美赛/国赛特等奖（O奖）和一等奖论文，学习建模思路、写作技巧和图表呈现。

## 分析模板

每篇分析论文建议包含：

```markdown
# [年份] [赛事] [题目] — [奖项]

## 基本信息
- 队号/学校
- 使用的主要方法
- 论文亮点（1句话）

## 建模思路摘要
（他们是怎么拆解问题的？用了什么模型？为什么选这个模型？）

## 值得学习的点
1. ...
2. ...
3. ...

## 可以改进的点
1. ...
2. ...

## 可复用的图表/写作技巧
...

## 关键公式/代码片段
...
```
