## 推荐笔记内容

### model-cheatsheet.md

当不确定用什么模型时，按以下顺序判断：

1. **问题是找最优方案？** → 优化模型（线性/整数/多目标/智能算法）
2. **问题是排顺序？** → 评价模型（AHP/TOPSIS/模糊综合评价）
3. **问题是预测未来？** → 预测模型（灰色/ARIMA/LSTM）
4. **问题是分类/识别？** → 分类模型（SVM/随机森林/XGBoost）
5. **问题是因果分析？** → 统计模型（DID/RDD/面板数据）
6. **问题是模拟复杂系统？** → 仿真模型（蒙特卡洛/系统动力学）
