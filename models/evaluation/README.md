# 评价与决策模型

## 总览

评价模型在数学建模竞赛中**几乎每届必出**，是出现频率最高的赛题类型之一（E/F题核心）。

## 模型列表

| 模型 | 文件 | 定权方式 | 适用场景 |
|------|------|---------|---------|
| 层次分析法(AHP) | `ahp.md` | 主观 | 分层评价体系、指标依赖性 |
| TOPSIS法 | `topsis.md` | 可主观/客观 | 多方案优劣排序 |
| 模糊综合评价 | `fuzzy_comprehensive.md` | 主观 | 定性指标多、边界模糊 |
| 灰色关联分析 | `grey_relational.md` | 客观 | 小样本、信息不完全 |
| 熵权法 | `entropy_weight.md` | 客观 | 数据充分、指标独立 |
| 主成分分析(PCA) | `pca.md` | 客观 | 降维、消除共线性 |
| 数据包络分析(DEA) | `dea.md` | — | 效率评价（CCR/BCC） |
| 秩和比法(RSR) | `rsr.md` | 非参数 | 非参数综合评价 |
| 因子分析 | `factor_analysis.md` | 客观 | 潜变量提取 |
| 组合评价 | `combined_evaluation.md` | — | 多方法综合对比验证 |

## 建模步骤（通用）

1. **建立评价指标体系** — 确定目标层、准则层、指标层
2. **指标预处理** — 正向化、标准化/归一化
3. **确定权重** — 主观/客观/组合
4. **综合评价** — 合成计算、排序
5. **结果分析** — 灵敏性分析（加分项）
