# 统计与计量模型

## 总览

统计模型是**全国大学生统计建模大赛**的核心要求，对国赛C题和美赛F题的政策分析也很重要。与传统数学建模不同，统计建模更强调**因果推断**和**统计显著性**。

## 模型列表

| 模型 | 文件 | 适用场景 | 推荐软件 |
|------|------|---------|---------|
| 面板数据分析 | `panel_data.md` | 多截面+多时段数据 | Stata/R/Python |
| 双重差分法(DID) | `did.md` | 政策效果评估 | Stata/Python |
| 断点回归(RDD) | `rdd.md` | 临界点政策效应 | Stata/R |
| 工具变量(IV) | `instrumental_variable.md` | 内生性处理 | Stata/R |
| 中介/调节效应 | `mediation_moderation.md` | 机制分析 | SPSS/R |
| 结构方程(SEM) | `structural_equation.md` | 潜变量+路径分析 | Amos/R(lavaan) |
| 空间计量 | `spatial_econometrics.md` | 地理数据空间依赖 | GeoDa/R |
| 生存分析 | `survival_analysis.md` | 时间-事件数据 | R(survival) |
| 合成控制法 | `synthetic_control.md` | 单一处理单元政策评估 | R(SCM) |

## 统计建模特殊要求

统计建模大赛与普通数学建模竞赛的关键差异：

| 维度 | 数学建模竞赛 | 统计建模大赛 |
|------|-------------|-------------|
| 选题方式 | 给定题目 | 给定主题，自拟题目 |
| 核心方法 | 优化/评价/预测为主 | 统计推断/因果/计量为主 |
| 软件 | MATLAB/Python | R/Stata/Python |
| 汇报形式 | 论文 | 论文+答辩PPT+答辩视频 |
| 字数限制 | 无严格限制（美赛25页） | ≤16,000字符 |
| 必需内容 | 模型+求解+分析 | 假设检验+p值+置信区间+稳健性 |
| 论文查重 | 各赛不一 | 知网查重≤20% |

```
    ┌───────┼───────┐
    ▼       ▼       ▼
  相关   自然实验  随机实验
  性     准实验     (金标准)
  │       │         │
 Pearson DID/RDD   RCT
 回归    工具变量   随机分配
```
