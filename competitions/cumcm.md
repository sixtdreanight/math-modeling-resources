# CUMCM 国赛指南

## 基本信息

| 项目 | 详情 |
|------|------|
| 全称 | 高教社杯全国大学生数学建模竞赛 |
| 主办方 | 中国工业与应用数学学会 |
| 时间 | 每年 9 月中旬（3天3夜） |
| 报名费 | ¥300/队（学校统一报名） |
| 参赛对象 | 本科生（有专科组） |
| 语言 | 中文 |
| 官网 | https://www.mcm.edu.cn |

## 题型分析

| 题目 | 难度 | 典型特征 | 选做人数 | 核心方法 |
|------|------|---------|---------|---------|
| **A题** | 较高 | 偏物理/工程，专业性强 | 较少 | 机理分析、微分方程、数值模拟 |
| **B题** | 中等 | 优化决策，题型不固定 | 中等 | 优化模型、图论、算法设计 |
| **C题** | 较低 | 数据分析，开放度高 | **最多** | 统计建模、机器学习、评价方法 |

> C题选的人最多（通常占总参赛队的 60%+），竞争也最激烈。A题选的人最少但物理/工程基础扎实的队容易出彩。

## 三天时间规划

```
第一天：
  08:00-12:00 → 选题讨论，确定方向（最关键！）
  12:00-18:00 → 查阅文献、理解问题
  18:00-24:00 → 建立初步模型框架
第二天：
  上午 → 模型完善，开始求解
  下午 → 核心模型求解完成
  晚上 → 开始论文写作
第三天：
  上午 → 补充分析和图表
  下午 → 完善论文、格式调整
  20:00 → 提交截止！
```

## 与美赛对比

| 维度 | 国赛 | 美赛 |
|------|------|------|
| 时间 | 9月（3天） | 1月（4天） |
| 语言 | 中文 | 英文 |
| 页数 | 无严格限制 | 25页（2026新规） |
| A题特征 | 中国工程背景 | 物理/连续 |
| AI使用 | 未明确 | 允许（需声明） |
| 公式标准 | 国标 | IEEE/自行设定 |
| 论文格式 | 有官方模板 | 自行排版 |

## 论文要点

- 摘要：500-800字，包含问题重述、方法概述、主要结果、结论
- 一定要做灵敏性分析和稳健性检验
- 图表清晰，标注中英文对照更佳
- 参考文献使用国标 GB/T 7714 格式
