# 全国大学生统计建模大赛指南

## 基本信息

| 项目 | 详情 |
|------|------|
| 主办方 | 中国统计教育学会 |
| 时间 | 每年 5 月（主题提前发布） |
| 报名费 | **免费** |
| 参赛对象 | 本科生、研究生 |
| 含金量 | ★★★★（国家级 A 类赛事） |

## 核心区别：主题制 vs 题目制

统计建模大赛**不给定固定题目**，而是给定**年度主题**，参赛队自行拟定研究题目。

| 年份 | 主题 |
|------|------|
| 2026 | "服务国家战略 创新统计赋能" |
| 2025 | "统计创新应用 数据引领未来" |

### 六大方向

1. 高质量发展与新质生产力统计监测
2. 数据要素市场化与数字经济统计创新
3. 双碳目标与绿色发展统计建模
4. 乡村振兴与共同富裕统计赋能
5. 宏观经济与风险防控统计预警
6. 科技自立自强与创新驱动统计研究

## 方法论核心

统计建模大赛**侧重统计推断和因果分析**，与数学建模竞赛的方法论有本质区别：

| 数学建模 | 统计建模 |
|---------|---------|
| 优化 → 最优解 | 估计 → 因果关系 |
| 预测精度 | 统计显著性（p值） |
| 灵敏性分析 | 稳健性检验（多种模型对比） |
| MATLAB/Python | R/Stata/Python |

### 必备方法

| 方法 | 用途 | 推荐软件 |
|------|------|---------|
| DID（双重差分） | 政策效果评估 | Stata/R/Python |
| RDD（断点回归） | 临界点政策效应 | Stata/R |
| 面板数据分析 | 固定/随机效应 | Stata |
| 工具变量 (IV) | 内生性处理 | Stata/R |
| 中介效应 | 机制分析 | SPSS/R |
| 合成控制法 | 单案例政策评估 | R |

## 论文要求

| 指标 | 要求 |
|------|------|
| 字数 | 正文字符 ≤ 16,000（含空格） |
| 查重 | 知网 ≤ 20%（超过 40% 学术不端） |
| 提交 | 论文 + 查重报告 + 答辩PPT + 答辩视频 + 数据包 + 代码 |
| AI使用 | 允许辅助（文献检索/润色），严禁生成正文 |

## 选题公式

**选题 = 问题 + 数据 + 方法**

- A类（问题+方法）：61%，如"数字经济对碳排放的影响测度"
- B类（地区嵌入）：15%，如"黄河流域高质量发展水平测度"
- D类（修辞型）：9%，如"'绿水青山就是金山银山'——森林碳汇研究"
