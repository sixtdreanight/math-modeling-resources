# MathorCup 妈妈杯指南

## 基本信息

| 项目 | 详情 |
|------|------|
| 全称 | MathorCup 高校数学建模挑战赛 |
| 主办方 | 中国优选法统筹法与经济数学研究会 |
| 时间 | 每年 4 月中旬（4天） |
| 报名费 | ¥200/队 |
| 报名平台 | 赛氪网 |
| 官网 | https://www.saikr.com/mathorcup |

## 核心特点

与其他数模竞赛最大的不同：**赛题来自企业实际问题**。合作企业包括阿里、滴滴、京东、华为等，题目更贴近真实商业场景。

### 赛题特征

| 维度 | 说明 |
|------|------|
| 题源 | 企业实际业务问题转化 |
| 题型数 | 通常 4 题（A/B/C/D） |
| 数据类型 | 经常提供真实企业数据 |
| 评分重点 | 方案的可行性和可落地性 |
| 奖项 | 全国奖（一/二/三等）+ 企业专项奖 |

### 常见赛题方向

| 方向 | 典型场景 | 对应模型 |
|------|---------|---------|
| 供应链优化 | 库存管理、路径规划、配送优化 | 整数规划、VRP、动态规划 |
| 调度排班 | 客服排班、配送调度、人员分配 | 整数规划、排队论、智能算法 |
| 推荐系统 | 用户画像、商品推荐、定价策略 | 协同过滤、聚类、回归 |
| 资源分配 | 带宽分配、仓储选址、投资组合 | 线性/非线性规划、博弈论 |

## 与国赛/美赛对比

| 维度 | MathorCup | 国赛 | 美赛 |
|------|-----------|------|------|
| 题目实用性 | 最高（企业真实问题） | 中 | 中 |
| 数学深度 | 中 | 高 | 高 |
| 获奖率 | ~50% | ~40% | ~35% |
| 企业认可度 | 高 | 高 | 最高 |
| 语言 | 中文 | 中文 | 英文 |

## 备赛建议

1. **重视方案可行性** — 企业题更看重落地，不是模型越复杂越好
2. **熟悉运筹优化方法** — 供应链、调度是高频方向
3. **代码可运行性** — 建议提交可直接运行的代码
4. **关注往年赛题** — 了解企业题的实际风格
5. **提前学习 PuLP/OR-Tools** — 整数规划求解工具
